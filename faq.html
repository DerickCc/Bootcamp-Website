Kamu minimal harus berusia 17 tahun dan maksimal 55 tahun untuk mengikuti Bootcamp Coddeng .

Tentu saja karena kami tidak menyediakan komputer di tempat kami. Operating System dari laptop tersebut dapat berupa Mac OS, Linux, atau Windows.

Untuk sekarang hanya tersedia pengajaran dalam bahasa Indonesia.

Biaya setiap Bootcamp Coddeng bervariasi dan akan berbeda pula jika dibayar dengan ataupun tanpa cicilan. Silakan mengecek di halaman kursus untuk mengetahuinya secara lebih detail.

Kamu akan mempelajari beberapa konsep di balik pengembangan aplikasi seluler. Mari kita perjelas: Kamu tidak akan mempelajari pengembangan iOS, Android atau Windows Phone. Kami benar-benar berpikir keterampilan ini tidak dapat diajarkan (jujur) dalam 9 minggu. Kami lebih suka mengajari Kamu dasar-dasar pemrograman dan pengembangan web karena berbagai alasan. Pertama, ini adalah keterampilan inti dari seorang insinyur perangkat lunak, bahkan jika Kamu ingin terjun ke pengembangan seluler nanti. Kedua, web adalah platform terbuka yang perlu Kamu pahami. Berkat Desain Web Responsif (yang merupakan bagian dari program), dan program mini WeChat (di Cina), Kamu akan dapat membuat aplikasi yang akan ditampilkan dengan baik di perangkat seluler. Kamu juga akan belajar membuat API yang akan digunakan oleh aplikasi seluler Kamu. Maka akan sangat mudah bagi pengembang seluler mana pun untuk membuat kode antarmuka iOS atau Android Kamu mulai dari API yang Kamu buat.

Kami tidak menyediakan akomodasi atau katering. Harga kursus hanya termasuk biaya pembelajaran - serta teh dan kopi tanpa batas. Meskipun demikian, pengelola kota kami akan dengan senang hati merekomendasikan beberapa tempat yang bagus untuk kamu menginap karena mereka biasa membantu siswa menemukan akomodasi dan memiliki banyak rencana bagus.

Benar, materi dapat kamu akses sampai kapan pun.
