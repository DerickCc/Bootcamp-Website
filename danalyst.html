Inilah Program untuk Upgrade Karirmu

Untuk menjadi web developer siap kerja, dibutuhkan waktu 10 bulan dan 15 jam seminggu

1. Sesi Onboarding

Apa yang kamu bisa ekspektasi sebagai siswa Coddeng. Bagaimana sprint terstruktur, siapa tim bimbingan kami, bagaimana cara menggunakan Slack, di mana mencari informasi dan lain-lainnya.

2. Python Dasar -- 3 minggu

Perkenalan kamu dengan dunia ilmu data! Konsep kunci dan sintaks dasar dalam Python. Loop, kondisi, dan fungsi. Library Pandas untuk analisis data. Studi kasus analitis pertamamu, diikuti dengan proyek pertamamu.

3. Pra-pemrosesan Data -- 3 minggu

+1 proyek ke portofoliomu

Mengompensasi data yang kurang sempurna. Menangani nilai yang hilang dan nilai duplikat. Mengubah tipe data. Pemikiran sistem untuk analis.

4. Analisis Data Eksploratif (EDA) -- 3 minggu

+1 proyek ke portofoliomu

Melakukan pemindaian awal untuk mendeteksi pola dalam data. Membuat grafik dasar dan menghasilkan hipotesis pertamamu.

5. Analisis Data Statistik -- 3 minggu

+1 proyek ke portofoliomu

Teori probabilitas, distribusi yang paling umum, dan metode statistik dalam Python. *Sampling* dan signifikansi statistik. Mengidentifikasi dan menangani anomali.

6. Proyek Terintegrasi 1 -- 1 minggu

+1 proyek ke portofoliomu

Mengidentifikasi pola untuk membantu kamu menentukan apakah permainan video tertentu akan sukses secara komersial atau tidak.

7. Pengumpulan dan Penyimpanan Data (SQL) -- 2 minggu

+1 proyek ke portofoliomu

Memahami struktur basis data dan bagaimana cara menarik data dari basis data menggunakan kueri SQL. Menemukan data secara daring.

8. Analisis Bisnis -- 2 minggu

+1 proyek ke portofoliomu

Penerapan analisis data dalam bisnis. Metrik bisnis dan Indikator Kinerja Utama. Analisis data pengguna. Analisis pemasaran dan instrumen terkait. Corong penjualan.

9. Pengambilan Keputusan Bisnis Berdasarkan Data -- 2 minggu

+1 proyek ke portofoliomu

Metode dan instrumen untuk menguji hipotesis. Desain eksperimental. Musiman (Seasonality). Analisis Kohort. A/B testing.

10. Cara Bercerita Menggunakan Data -- 2 minggu

+1 proyek ke portofoliomu

Menyajikan hasil risetmu. Metode visualisasi data. Membuat laporan untuk menjelaskan temuan. Library Seaborn.

11. Proyek Terintegrasi 2 -- 1 minggu

+1 proyek ke portofoliomu

Menyatukan semua yang telah kamu pelajari sejauh ini untuk menganalisis hasil A/B testing dari sebuah aplikasi makanan.

12. Otomatisasi -- 2 minggu

+1 proyek ke portofoliomu

Mengotomatiskan proses analisis data dan membuat script untuk tugas rutin. Metode visualisasi data. Mempresentasikan hasil.

13. Ramalan dan Prediksi -- 2 minggu

+1 proyek ke portofoliomu

Metode dan penerapan machine learning. Klasifikasi, peramalan, pengklasteran. Regresi. Decision tree.

14. Proyek Akhir -- 2 minggu

+1 proyek ke portofoliomu

Menerapkan semua yang telah kamu pelajari dalam pelatihan dua minggu yang menyimulasikan pengalaman kerja sebagai seorang analis data junior.

15. Program Karier -- 10 minggu

Selain kursus pendidikan utama, jalur karier meliputi panduan tertulis, tugas pekerjaan rumah, tool yang berguna, dan kolaborasi model peer-to-peer. Setiap bagian (atau sprint) berlangsung dua minggu dan memerlukan sekitar 8 jam belajar.
